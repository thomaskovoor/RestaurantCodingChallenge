The project aims to generate a solution for listing all the food options available for a person on a particular area of the map.
The user should be able to view all the restaurants after choosing country and city and filter them according to his choice.The restaurants
should be displayed according to the order of their ratings and one should be able to filter them according to online booking,online delivery and cuisines.

The solution i have though of is to accept the country name and the city name through an html form and then display the restaurants based on the selection.The country code and city code can be used to seach through the data to display the restaurants.
